# 🎧 TWS 耳机控制 APP

基于 **Electron + React + AB-Mate 协议**的轻量级 TWS 耳机控制应用

## ✨ 功能特点

- ✅ **10段 EQ 均衡器** - 专业音效控制
- ✅ **ANC 主动降噪** - 4档降噪 + 3档透传
- ✅ **游戏模式** - 低延迟连接
- ✅ **设备查找** - 蜂鸣定位
- ✅ **实时电量** - 左右耳 + 充电盒
- ✅ **音量控制** - 实时调节

## 🚀 快速开始

```bash
# 1. 安装依赖
npm install

# 2. 启动开发
npm run dev
```
